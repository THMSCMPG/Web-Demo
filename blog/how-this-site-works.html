How This Site Works

April 20, 2025

This site is a static HTML site hosted on GitHub Pages. The pages, styles, and scripts live directly in this repository.

Why this setup

  • No monthly fees. GitHub Pages hosting costs nothing.
  • No CMS to log in to. Writing happens directly in this folder.
  • Fast. Just static HTML — no database, no server processing.
  • Safe. Every change is saved in Git history. You can always roll back.

The only things you ever touch

What you want to doWhat you do
Change site name / taglineEdit the shared header/title text in your HTML pages
Write a blog postCreate a new HTML page in /blog/
Add a projectCreate a new HTML page in /work/
Update your About pageEdit about.html
Add an imageDrop it in assets/img/

Everything else — the layout, navigation, styling, and all the code — is managed separately and updates automatically.

What happens after you commit

GitHub Pages serves the updated static files directly. The live site updates quickly after deployment.
