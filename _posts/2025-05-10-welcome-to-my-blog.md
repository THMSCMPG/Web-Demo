## How to write your own posts

1. Duplicate this file in the `_posts/` folder.
2. Rename it following the `YYYY-MM-DD-your-title.md` pattern — the date in the name is what Jekyll uses for ordering.
3. Update the front matter (the block between `---` lines) at the top.
4. Write your post in Markdown below.
5. Save and commit — your post is live within about a minute.

## Formatting cheatsheet

**Bold text** — wrap in `**double asterisks**`

*Italic text* — wrap in `*single asterisks*`

[A link](https://example.com) — `[link text](url)`

### Headings

Use `##` for a section heading, `###` for a subheading.

### Images

Drop an image into `/assets/img/` and reference it like this:

```
![Description of image](/assets/img/your-image.jpg)
```

### Lists

- Just start a line with a dash
- For a numbered list, use `1.`, `2.`, etc.

### Code

Use backticks for `inline code` or triple backticks for a code block:

```
your code here
```

That's everything you need to know.
